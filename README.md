# تطبيق إدارة مخازن مع تتبع المخزون والطلبات
==============================================

## نظرة عامة وغرض المشروع
------------------------

تطبيق إدارة مخازن مع تتبع المخزون والطلبات هو نظام إدارة مخازن متكامل يسمح للمستخدمين بإدارة مخازنهم بفعالية، بما في ذلك تتبع المخزون والطلبات. يهدف هذا المشروع إلى توفير حل متكامل ومرن لإدارة المخازن، مما يسمح للمستخدمين بتحسين كفاءة عملياتهم وتقليل التكاليف.

## هيكل المشروع
----------------

* `app`: تطبيق الويب الرئيسي
* `database`: قاعدة البيانات
* `docker`: ملفات Docker
* `requirements`: متطلبات التطبيق

## تشغيل البيئة
----------------

لتشغيل التطبيق، اتبع الخطوات التالية:

1. قم بتثبيت Docker و Docker Compose على جهازك.
2. قم بتشغيل الأمر `docker-compose up` في المجلد الرئيسي للمشروع.
3. انتظر حتى يتم تشغيل التطبيق و قاعدة البيانات.
4. افتح متصفح الويب و اذهب إلى `http://localhost:8000` لاستخدام التطبيق.

## قائمة الوحدات والجداول والأدوار
---------------------------------

### الوحدات

* `users`: وحدة المستخدمين
* `products`: وحدة المنتجات
* `orders`: وحدة الطلبات
* `inventory`: وحدة المخزون

### الجداول

* `users_table`: جدول المستخدمين
* `products_table`: جدول المنتجات
* `orders_table`: جدول الطلبات
* `inventory_table`: جدول المخزون

### الأدوار

* `admin`: دور المسؤول
* `user`: دور المستخدم العادي
